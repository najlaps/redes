1.1 Primeiro foi criado uma interface do tipo monitor, depois utilizar um comando (airodump-ng) para capturar beacons na interface criada.
A interface monitor poder capturar tudo e quebrar senhas.

1.2 A fase de geração e distribuição de chaves começa com o 4-Way Handshake que permite que o ponto de acesso e a estação derivem a PTK( pairwise transient key) e realizem uma autenticação mútua.
O host envia mensagem solicitando uma conecxão, o ap envia uma mensagem para o host, o ap envia uma mensagem de desafio para o host e o mesmo responde.
